# Changelog

## v1.0.0 - 2026-02-22

- Alta rapida simplificada para carga diaria (tipo, monto, categoria y guardar).
- Modo planilla avanzada editable para carga y edicion masiva.
- Exportacion de reportes en XLSX, CSV y backup completo en JSON.
- Categorias personalizadas para adaptarse a cada usuario.
- Presupuestos por categoria con seguimiento mensual.
- Vencimientos y recordatorios de pago con alertas in-app.
- Mejoras UX mobile/iPhone: foco, teclado, legibilidad y jerarquia visual.
